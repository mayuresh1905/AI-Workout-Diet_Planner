# 🏋️ AI Fitness Coach

An AI-powered fitness and meal planning web application built using
Python and Google Gemini API.

## 🚀 Features

- Workout plan generator
- 7-day AI meal planner
- Meal image analysis
- Personalized responses using Gemini API

## 🛠 Tech Stack

- Python
- Streamlit
- Google Generative AI (Gemini)
- Hugging Face Deployment

## 🔐 Environment Variable

Set your API key as:

GEMINI_API_KEY = your_api_key_here

## 📦 Installation

pip install -r requirements.txt
streamlit run app.py
